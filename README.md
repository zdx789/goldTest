# goldTest

Cocos Creator 2.4.11 shader extraction project.

## Contents

- `assets/effects/` — Extracted .effect shader files
- `assets/materials/` — Material parameter references  
- `scripts/` — Browser console tools for runtime extraction
- `tools/` — Browser-console debug tools (mesh export, material inspection, light debugger, effect dumper)

See [`tools/README.md`](tools/README.md) for full documentation on the debug tools.
